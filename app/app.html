Instructions

  1. Select the input directory. This should be a folder containing only the images that need to be processed.
  2. Select the output directory. This will be the location where all the processed images will be placed when they are ready.
  3. Select the desired width. The images will be sized down to that width; the image aspect ratio will be preserved.
  4. Press the PROCESS IMAGES button once the above options have been set.
